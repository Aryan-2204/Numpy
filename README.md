# 🔢 Python NumPy Learning Journey

Welcome to my **NumPy learning repository**!

##  About NumPy

**NumPy (Numerical Python)** is a core Python library used for:

* Fast numerical computations
* Working with arrays and matrices
* Performing mathematical and statistical operations
